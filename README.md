# Real Estate Market App

Am application that simulates an online Real Estate Market developed with NextJS and Sanity as primary tech stack. Offers powerful features such as:

- posibility to add, edit and delete real estate properties directly from the Sanity CMS Studio
- view the available listings in the homepage
- login functionality, allowing saving properties to your account to be able to view them later and compare to others
- billing for real estate agents, who have additional options such as listing properties, viewing dashboard with stats, etc
- possibility to contact agent to see a certain house
- advanced searching for finding the ideal home to buy

## **Includes the following main tech:**

| Technology                | Version       |                                                                             |
| ------------------------- | ------------- | --------------------------------------------------------------------------- |
| **Next.js**               | 16.x.x        | Full-stack React framework with App Router                                  |
| **Sanity CMS**            | 4.x.x         | Headless CMS for structured content, real-time editing, and API-driven data |
| **React**                 | 19.x.x        | Component-based UI library with hooks and modern concurrent rendering       |
| **Clerk**                 | 7.x.x         | Authentication, user management, and social login + billing                 |
| **Tailwind CSS**          | 4.x           | Utility-first CSS framework for UI styling                                  |
| **shadcn/ui**             | 4.x.x         | Accessible, customizable out-of-the-box UI components                       |
| **React Hook Form + Zod** | 7.x.x / 4.x.x | Performant form handling with type-safe schema validation                   |
